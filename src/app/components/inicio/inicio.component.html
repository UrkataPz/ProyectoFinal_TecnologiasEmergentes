Problema identificado

Muchos estudiantes no conocen qué clubes existen, quiénes los integran, qué eventos realizan o cómo pueden formar parte de ellos.

Objetivo del sistema

Crear una base funcional para una aplicación web que permita consultar clubes, navegar entre secciones y preparar el desarrollo de futuras funciones.
